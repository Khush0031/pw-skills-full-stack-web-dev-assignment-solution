List of HTML Tags

Below is an ordered list of HTML tags and their Information:

  1. <html>: Defines the root of an HTML document.
  2. <head>: Contains metadata about the document.
  3. <title>: Sets the title of the web page.
  4. <meta>: Provides metadata about the HTML document.
  5. <link>: Defines relationships with external resources.
  6. <style>: Contains CSS for styling the document.
  7. <script>: Embeds or references JavaScript code.
  8. <body>: Contains the visible content of the web page.
  9. <h1> to <h6>: Headings of varying importance.
  10. <p>: Represents a paragraph of text.
  11. <a>: Defines a hyperlink to other web pages or resources.
  12. <img>: Embeds an image in the web page.
  13. <ul>: Creates an unordered (bulleted) list.
  14. <ol>: Creates an ordered (numbered) list.
  15. <li>: Represents an item in a list.
  16. <div>: A generic container for grouping and styling elements.
  17. <span>: A generic inline container for styling text.
  18. <form>: Defines an interactive form for user input.
  19. <input>: Represents an input field in a form.
  20. <button>: Creates a clickable button within forms.
