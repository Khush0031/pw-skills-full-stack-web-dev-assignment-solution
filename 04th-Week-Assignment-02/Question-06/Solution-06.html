HTML vs. HTML5

HTML HTML5
HTML stands for HyperText Markup Language. HTML5 stands for HyperText Markup Language 5.
HTML is the older version of the markup language. HTML5 is the latest version of the markup language.
HTML4 and earlier versions had limited support for multimedia and interactive elements. HTML5 provides native support for audio, video, canvas, and other multimedia elements.
HTML4 used presentational elements and attributes for styling and layout. HTML5 encourages the use of CSS for styling and removes many presentational elements.
HTML documents require more code for the same functionality (e.g., doctype declaration). HTML5 simplifies document structure with a shorter doctype declaration (<!DOCTYPE html>).
HTML4 had limited support for client-side storage. HTML5 introduces local storage and session storage for client-side data storage.
HTML4 did not provide specific elements for headers, footers, and navigation sections. HTML5 introduces semantic elements like <header>, <footer>, <nav>, and others for better document structure.
